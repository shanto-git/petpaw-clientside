# PetPaw:

A simple React + Firebase petspaw where users can browse pets, add listings, and manage their own posts.

## Live URL
https://assignment-10-fc740.web.app

## Purpose
- Browse pets & supplies  
- Add new listings (logged-in users)
- Manage personal listings & orders
- Simple authentication system

## Features
- Firebase Login / Register / Logout  
- Dynamic Navbar (before & after login)  
- Add Listing / My Listings / My Orders  
- Responsive UI with Tailwind

## Tech Stack
- React  
- React Router  
- Firebase Auth  
- Tailwind CSS  
- React Icons
- Framer motion
